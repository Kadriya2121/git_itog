# Инструкция по работе с командами Git


![git основные команды](course_12618_image.png)

## 1. Настройка учётной записи
Перед началом работы укажите имя пользователя и email (эти данные будут привязаны к вашим коммитам):

**git config --global user.name "Ваше Имя"**
**git config --global user.email "ваш_email@example.com"**

## 2. Создание и подключение репозитория

Инициализировать новый репозиторий в текущей папке

**Git init**  

## 3. Работа с изменениями
Проверить статус файлов (какие изменены, добавлены или удалены):
**git status** 

> ***Добавить файлы в индекс (подготовить к коммиту):***
1. Один файл: git add имя_файла
2. Все изменённые файлы: git add .
3. Все файлы (включая новые): git add :/

## Сделать коммит (зафиксировать изменения):

**git commit -m "Краткое описание изменений"**

## Посмотреть историю коммитов:

Полный список: **git log**

Упрощённый формат: **git log --oneline**
